# playwright-automation-framework
Modern Test Automation Framework using Playwright, API Testing, Database Validation and GitHub Actions

## Overview

Enterprise-grade test automation framework built using Playwright and TypeScript.

## Technologies

- Playwright
- TypeScript
- Node.js
- GitHub Actions

## Features

- UI Automation
- API Testing
- Database Validation
- CI/CD Integration

## Installation

```bash
npm install
```

## Run Tests

```bash
npx playwright test
```
